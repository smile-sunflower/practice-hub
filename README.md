## Repository Structure(仓库的通用结构)

仓库通用目录结构分为以下格式：

**book**（该技术的相关书籍，slides等信息）

​				--------**slides**(比较好的slides)

**conference**（该技术的行业会议）

**course** （该技术的课程）

**paper** （该技术相关的论文）

**wikipedia** （该技术的维基百科，视频链接，awesome，相关人物，笔记等信息）

​				--------**notes**（自己学习的笔记整理）

**daily** （该技术某一天的学习）

​				--------**yyyy-MM-dd.md**

**code**（该技术相关代码）
